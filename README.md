# Chat SDK Android Demo

This demo project shows a simple example of how to integrate the Chat
SDK into an Android application using Kotlin and Flow.

The SDK itself is callback-based to support both Java and Kotlin
projects. This demo demonstrates how to adapt the SDK for Kotlin using
coroutines and StateFlow.

## Features demonstrated

-   SDK initialization  
-   Token-based and ContactIdentity authentication  
-   Loading contacts  
-   Loading dialogs  
-   Sending messages  
-   Receiving real-time updates  
-   Optimistic UI updates  
-   Connection state handling  
-   Push notifications (FCM)  
-   Simple local persistence using SharedPreferences
